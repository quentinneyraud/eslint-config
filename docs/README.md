# MYLIB

> View doc [here](http://MYLIB.com)

## Install

```bash
yarn add MYLIB 
```

## Usage

```js
import MYLIB from 'MYLIB'

```

## Development

```
# clone repo
git clone git@github.com:quentinneyraud/MYLIB.git
cd MYLIB

# install dependencies
yarn

# Run example
npm run dev
```

## Build

```bash
npm run build
```
